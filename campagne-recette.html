Structure attendue

La premiere ligne utile de votre feuille doit contenir les en-tetes de colonnes. Les colonnes ci-dessous sont attendues (les noms exacts ne sont pas obligatoires, les variantes Jira sont egalement reconnues) :

Identificateur de cas de test Identifiant unique — variante : ID
Priorité Haute / Moyenne / Basse ou P1 / P2 / P3
Données Profil testeur (optionnel) — ex: Admin, User Manager... — variante : Role
Résumé Intitule du cas de test — variante : Scenario
Action Etape(s) a executer — variante : Etapes
Résultat Attendu Comportement attendu — variante : Attendu
Chemin de la bibliothèque de tests Groupe les tests par bibliothèque Xray — variante : Bibliotheque de tests
